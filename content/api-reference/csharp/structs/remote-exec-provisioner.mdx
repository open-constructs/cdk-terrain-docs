The remote-exec provisioner invokes a script on a remote resource after it is created.

This can be used to run a configuration management tool, bootstrap into a cluster, etc
The remote-exec provisioner requires a connection and supports both ssh and winrm.

See [remote-exec](https://developer.hashicorp.com/terraform/language/resources/provisioners/remote-exec)

## Initializer <a name="Initializer" id="cdktn.RemoteExecProvisioner.Initializer"></a>

```csharp
using Io.Cdktn;

new RemoteExecProvisioner {
    string Type,
    SSHProvisionerConnection|WinrmProvisionerConnection Connection = null,
    string[] Inline = null,
    string Script = null,
    string[] Scripts = null
};
```
