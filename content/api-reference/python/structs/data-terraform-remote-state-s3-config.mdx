### `no_proxy`<sup>Optional</sup> <a name="no_proxy" id="cdktn.DataTerraformRemoteStateS3Config.property.noProxy"></a>

```python
no_proxy: str
```

- *Type:* str

(Optional) Comma-separated list of hosts that should not use HTTP or HTTPS proxies.

Each value can be one of:

- A domain name
- An IP address
